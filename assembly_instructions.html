Assembly Instructions

Parts

  • 2 0.1uF SMD Capacitor
  • 2 10uF SMD Capacitor
  • 1 10x1 Female Pin Header
  • 2 16x1 Female Pin Header
  • 1 1k SMD Resistor
  • 2 2k SMD Resistor
  • 3 300 SMD Resistor
  • 2 35pF SMD Capacitor
  • 1 50 SMD Resistor
  • 1 7x1 Female Pin Header
  • 1 INA592 Diff Amp
  • 1 LM358 OP Amp
  • 1 REF200 Current Source
  • 6 SPDT Switch

Tools

  • 1 Curved Soldering Iron Tip
  • 1 Grabber Tool
  • 1 Soldering Iron
  • 1 Straight Soldering Iron Tip

Step 1: Order PCB

  1. Download the Gerber File, BOM and Pick and Place File for the PCB.
  2. Sign into account or create an account if new.
  3. Go to JLCPCB Website website and upload Gerber zip folder.
  4. While on the page, drag and drop Gerber zip folder into area that asks for it. In the example image, I also used the SMT assembly service by selecting it.
  5. Upload BOM and Pick and Place .csv files. Select electronics and hobbies/developmental board for the description.
  6. Select parts that you desire to be soldered onto the board for you via the SMT assembly service.

Note SMT Assembly service is an affordable service where certain components are soldered to the board for you. Not every current component is compatible with this service. I recommend using this as its time consuming to solder these very tiny components.

Step 2: Order Components

  1. Go to mouser project and order integrated circuit (IC) components.
  2. Order remaining generic parts such as 603 footprint resistors and capacitors.

Note If you used the SMT assembly service, you do not need to order those parts again.

Note Not all parts make sense to order individually. I recommend ordering the 603 footprint resistors and capacitors as part of variety book set.

Step 3: Solder Surface Mount Components (SMD)

Solder components onto circuit board in the order below with a Soldering Iron. A strategy that has worked well with small chips is to use a soldering iron with very fine 0.3mm solder. First place some solder onto each pad for the component, then place component with Grabber Tool and press into place with tweezers and gently touch each lead from the component with the soldering iron until the solder melts around it.

Note a Curved Soldering Iron Tip will help solder SMD components to board.

Note If you used the SMT service, some of these parts may already be installed.

  1. 10uF SMD Capacitor
  2. 35pF SMD Capacitor
  3. 0.1uF SMD Capacitor
  4. 2k SMD Resistor
  5. 50 SMD Resistor
  6. 300 SMD Resistor
  7. 1k SMD Resistor
  8. SPDT Switch
  9. INA592 Diff Amp
  10. REF200 Current Source
  11. LM358 OP Amp

Step 4: Solder Through Hole Components

Note a Straight Soldering Iron Tip will help solder through hole components to board.

  1. 2x 16x1 Female Pin Header, one for each side of Arduino Nano IOT
  2. 1x 7x1 Female Pin Header in ad9833 socket
  3. 1x 10x1 Female Pin Header in ADS1115 socket

Step 4: Solder Male Pin Headers

If ordered any breakout boards (ADS1115, AD9833, Arduino, etc.), solder male pin headers to each board.

Step 5: Attach Breakout Boards

Attach each breakout board and Arduino into respective sockets.

Step 6: Flash Arduino Sketch

Currently no sketch is available.
